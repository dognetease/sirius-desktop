三、完善您团队和成员信息

3.1 完善您团队的部门

您可以在管理后台，创建并管理您团队下的部门。

  • 登录管理后台后，先点击左侧tab下的 「组织机构管理-部门管理」。

  • 在「部门管理」页面，点击对应的部门，在右侧可以进行部门名称，下级部门和部门下新增账号的操作。

3.2 为您团队的成员创建账号

您可以在管理后台，创建并管理您团队成员的账号。

  • 创建账号：登录管理后台后，点击左侧tab下的 「组织机构管理-账号管理」。

  • 在账号管理页面，点击新建账号，可以创建您团队成员的账号。

  • 账号管理：在账号管理页面，点击某个账号，进入账号设置页面。在此页面，可以重设账号密码，设置账号的部门等。
