什么是钓鱼邮件
钓鱼邮件是指利用电子邮件欺骗收件人将账号、密码等信息回复给指定的接收者；或引导收件人点击网页链接（如伪装的网上银行登录页）并输入银行卡号码、账户名称及密码等信息，从而造成隐私信息泄露或直接经济损失。
怎么识别钓鱼邮件
经系统识别，该邮件疑似钓鱼邮件，建议您仔细核查、提高警惕。您可以通过以下方式核实是否为钓鱼邮件:
1.看发信地址
钓鱼邮件常通过不易识别的错误拼写骗取收件人信任。如钓鱼邮件的发信地址是admin@offlce163com，而实际企业的真实域名是@office163com。
2.看邮件标题及附件
典型的钓鱼邮件标题通常包含“邮件投递失败、邮箱账号密码到期、邮箱容量上限、新冠防疫时事”等，遇到这类邮件，尤其需要警惕、核实其他细节内容。在下载邮件附件前，应仔细检查附件文件名和格式，不要因好奇下载可疑附件。
3.看正文内容及措辞
正文中包含链接或二维码、或索要登录账号密码等隐私信息，一定要提高警惕，通过邮件点开的网址不要轻易输入个人隐私信息。对通过“紧急、失效、重要、尽快"等词制造紧急气氛的邮件需谨慎辨别，不要忙中犯错。
